## 🚀 How to Run the Scripts

1. Clone the repository:
```bash
git clone <your-repo-link>
cd <repo-folder>
```
2 Make scripts executable:
```bash
chmod +x *.sh
```
3. Run the scripts:
```bash
./script1.sh
./script2.sh
./script3.sh
./script4.sh <logfile> <keyword>
./script5.sh
```
